# マネジャーの全仕事

## はじめに
- 「チームが機能すれば、個人でやるより優れた成果を出せる」。
- 「考え抜かれた行動」と「品格ある振る舞い」この2つを守って公開することはない。

## 第一部マネジャーになるあなたへ
- 出来るマネジャー
  - 人という非常に複雑なものの集まった組織を尊重し、理解して導くことが必要。

## 第一章マネジャーになるまで
- 管理職に登用されるまでのプロセスはさまざまだが、残念なことに、人材の見極めや登用の基準がきちんとしている企業はすくない。
- マネジメント能力とは、個人として成果を上げるためのスキルとはまったく別のものだ。
- マネジャーには、「一流の個人プレーヤーとしてのスキル」とは別種の能力が要求される。それが「人を動かす力」だ。
- チームとして最善になるように俯瞰して考えるのがマネジャーの仕事である。
  - こだわって職人的に仕事を進めればうまくいったプレーヤー時代とは大違いだ。

### 「マネジメントは誰でもできる」は間違い
- いまだに多くの企業では、管理職の育成が「出たところ勝負」になっている。
- そもそもマネジメントのやり方は教えなくてもわかるものだ、と思っているらしい。
  - 組織の明暗を分けるマネジメントであるので、なりゆき任せされているのは大問題だ。
  - 昇進してもその後うまくいかず、以前の業務に戻りたがっている人を職場で見かけないだろうか。
  - こういった不幸が起きているのは、管理職以外の昇進ルートが存在しないためだ。
  - 「管理職に不向きな人が起用されてしまう問題」をうまいやり方で解決した企業がある。
    - マネジメント未経験の管理職候補生たちが受ける研修の一環として、管理職の仕事が体験できる一日ワークショップを開いた。

### 「なんでも自分でできる人」がダメな理由
- 何でも自分でやった方が仕事がうまいく、と考えるタイプの人がいるが、こういう姿勢ではリーダーやマネジャーとしては、上手くいかない。
- 仕事を割り振って部下に任せるのが下手では管理職は失格だ。
- こうした管理職の下では、従業員の離職も起こりがちだ。「なんでも自分でできる人」が思っている以上に部下には能力があるもので、つまならない仕事ばかりやらされていると嫌になってしまうのだ。
