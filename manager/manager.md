# マネジャーの全仕事

## はじめに
- 「チームが機能すれば、個人でやるより優れた成果を出せる」。
- 「考え抜かれた行動」と「品格ある振る舞い」この2つを守って公開することはない。

## 第一部マネジャーになるあなたへ
- 出来るマネジャー
  - 人という非常に複雑なものの集まった組織を尊重し、理解して導くことが必要。

## 第一章マネジャーになるまで
- 管理職に登用されるまでのプロセスはさまざまだが、残念なことに、人材の見極めや登用の基準がきちんとしている企業はすくない。
- マネジメント能力とは、個人として成果を上げるためのスキルとはまったく別のものだ。
- マネジャーには、「一流の個人プレーヤーとしてのスキル」とは別種の能力が要求される。それが「人を動かす力」だ。
- チームとして最善になるように俯瞰して考えるのがマネジャーの仕事である。
  - こだわって職人的に仕事を進めればうまくいったプレーヤー時代とは大違いだ。

### 「マネジメントは誰でもできる」は間違い
- いまだに多くの企業では、管理職の育成が「出たところ勝負」になっている。
- そもそもマネジメントのやり方は教えなくてもわかるものだ、と思っているらしい。
  - 組織の明暗を分けるマネジメントであるので、なりゆき任せされているのは大問題だ。
  - 昇進してもその後うまくいかず、以前の業務に戻りたがっている人を職場で見かけないだろうか。
  - こういった不幸が起きているのは、管理職以外の昇進ルートが存在しないためだ。
  - 「管理職に不向きな人が起用されてしまう問題」をうまいやり方で解決した企業がある。
    - マネジメント未経験の管理職候補生たちが受ける研修の一環として、管理職の仕事が体験できる一日ワークショップを開いた。

### 「なんでも自分でできる人」がダメな理由
- 何でも自分でやった方が仕事がうまいく、と考えるタイプの人がいるが、こういう姿勢ではリーダーやマネジャーとしては、上手くいかない。
- 仕事を割り振って部下に任せるのが下手では管理職は失格だ。
- こうした管理職の下では、従業員の離職も起こりがちだ。「なんでも自分でできる人」が思っている以上に部下には能力があるもので、つまならない仕事ばかりやらされていると嫌になってしまうのだ。
- 「仕事を全部、自分でやらなくてはならないのは部下が引き受けてくれないからだ」と思い込んでいる。
- 「そういえば部下に小さい仕事しか割り振れていないな」と気づいたら立ち止まって、自分のマネジメントのやり方を見直してみよう。
- 「替えのきかない大事な人」としての自尊心もくずれて生きがいを失うためか、その後の人生が楽しくなさそうな人も多い。

### 経営者・上司とのコネで昇進した場合
- 世の中には「偉い人」とコネがあるとか、上司と親しいという理由でリーダーに選ばれる人もいる。
  - この手の企業で働いていないなら、自分を幸運だと思った方が良い。
- 優良な組織であれば、細かい知識の有無でマネジメント人材を選ばす、リーダーとしての資質に光るものがあるかで判断される。これは、自分で磨いていくしかない。
- リーダーシップを定義するのは難しい
  - リーダーとは進むべき方向を人々に指し示し、いつも適切な判断をすることで尊敬を勝ち得ている人のこと。
  - 合理的な判断を重ねて妥当な意思決定をする能力を磨いてい行く中で、リーダーとしての特性が身についていく。
  - 自らの意思決定に確信を持てるようになり、難しい決定もひるまずできるようになるのだ。
- リーダーとは
  - 未来を見通し、自分の意思決定による結果をはきっりと思い描ける人のことだ。
  - 自分の決定によって生じる感情的な反応に鈍くていいわけではなく、それに惑わされて意思決定を歪めないことが重要。

## 第二章 管理職としてのスタート
- マネジャーになって最初の一週間は、控えめに言っても「普通じゃない」。
- 人間行動学の学生が驚くような発見に満ちている。

### まずは慌てない
- 昇進を誰もが歓迎しているわけではない。
  - 同僚の中には「自分の方が適任だったのに」と思っている人もいる。
  - 役職を妬んで内心「失敗すればいいのに」と考えている人もいる。
- 逆に職場の「イエスマン」たちはイキナリ取り入ろうとしてくる。
  - 選ばれしあなたは、彼らにとって「成功への切符」だからだ。
  - 急に擦り寄ってくるのは気持ちがいいものではない。
- いきなり試してくる人もいる
  - 「答えられますか？」と質問をぶつけてくる。
- その他は、様子見の態度をとってくる。
  - 実際に動いて結果がでるまでは褒めも、貶しもしない。
- 前任者と比較される。
  - 前任者がダメ上司だった場合には、普通にやっているだけで高評価を得やすい。
  - 一方、優秀な人の後任となった場合、適応するのがなかなか厳しい。
- 新任の管理職の多くは上長とのコミュニケーションはうまくやれるが、直属の部下と上手くいかないようだ。
  - 自分の将来に影響が大きいのは上司よりも直属の部下の方である。
  - 今後の評価は部署の業績
    - 自分のチームがどれだけ上手く機能しているかで決まるから。
    - 自分のキャリアにとって最重要なのは直属の部下たちである。

### 新しい職権を振りかざさない
- 新しい役職に伴う権威については「数量限定の在庫」だと考えておきたい。
  - 使わずとっておけば本当に必要なときに大量に使えるわけだ。
- 権威を使わずしまっておくと本当に必要な事態では強い効果を発揮できる。
- 周囲が見慣れていないから。
  - 部下はあなたが上司だとわかっているし、あなたからの依頼には権限が伴うとも理解している。
  - わざわざ権威を振りかざす必要はない。
- 要するに地位があるからと言って権力を振りかざす必要はない。
  - **穏やかに部下に接していれば部下から悪い印象を持たれずに済む。**
  - **一度、悪いイメージができると、あとで消すのはほぼ無理なので、この点でも非常に重要である。**

### 個別に話す機会を持つ
- 着任後２ヵ月以内に自分の監督範囲にあるメンバー全員と個人面談をしておくとよい。
- 面談の際には、個室に呼ぶなりランチや社外でのお茶に誘うなどして部下がのびのびと思いを話せる状況を状況を作る。
- 部下の個人的な問題も大切であるが、面談の話題は仕事関係に限定しておいたほうがいい。
- 家庭の事情こそが最大の問題だという部下もいる。
  - それでも個人的なアドバイスはしないように。
  - 上司になったのであって、人生の問題解決のプロではない。
  - ただ、話を聴こう。誰かに話しを聴いてもらいたいだけのことも多い。
- 「メールや電話でも十分でしょう?」とは考えないように。論外だ。
- 対面とのコミュニケーションの質に大きな差があり、望むような関係性の構築は難しい。

### 部下を理解しよう
- 部下との個別面談の目的は、相手が今後、あなたとコミュニケーションしやすいように接点を作っておくことだ。
- 部下の関心ごとに心から興味を示し、一人ひとりのキャリア上の目標や野心を理解することも大切。
- 部下の価値観や考え方が引き出せるような質問をしよう。
  - うわべの関心を装うのではなく、部下が楽しく健やか働けるよう、本気で考えよう。
  - 着任後の面談の目的は「上司は部下一人ひとりのことを真剣に考えて、目標を実現できるようサポートしてくれる」と部下に分かってもらうことだ。
- マネジャーとなって数か月もしないうちに、自分の技術的なスキルは、人間力に比べればさして重要ではないと気付くだろう
- 逆に技術面では非常に優秀でも人間力が欠如している場合には、マネジメントは難しいはずだ。

### 友人が部下になっても特別扱いしない
- たとえ友人関係が続いても仕事上の接し方は以前と変えなければならない
- 新人マネジャーとしては友人関係の有無にかかわらず「この上司は部下とこんなふうに仕事をする」という印象を確立すべき。」
- 全ての部下に対して、同じ業績基準、行動基準を適用し等しく結果責任を取らせるべき。

### 組織の編成は慎重に
- 部下一人ひとりについて、その役割も含めて熟知しているのでない限り、組織の改編には慎重になるべきだ。
- 組織の再構築、すなわちリストラは関係者全員にとって強いストレスになる。
- 経験の浅いマネジャーは管理範囲を広げすぎて失敗しがちだ。やりたくなるのも無理もない
- ほとんどの部下は、あなたの直属の部下になりたいだろう。
- マネジャーが持てる直属の部下の数は限られていることだ。
  - 部下との物理的な距離。自分と同じ施設内にいるのであれば、少し多めの部下でも管理できる。
  - 目安として週1回、個別のミーティングを設定できる人数を直属で持つ部下の数の上限としておく。必ず1体1で顔を突き合わせてのミーティングができる人数。
  - 部下との個別定例は毎週、欠かささないよう気を付ける。

### 自分の気分をマネジメントする
- 仕事以外の問題のせいで、気分に影響がでてしまうことは誰にもある
- マネジメントの本にはよく「家庭の問題は職場に持ち込むな、逆もしかり」と書いてある
- まず、「ある原因で自分の心が乱れていて、その影響が仕事にもでているかもしれい」と受け入れいることだ。
- いっそ、「ご覧の通り、今日は気分が絶好調でないんだ。もしイライラしているように感じたなら本当に申し訳ない。」

### 自分の感情をマネジメントする
- 優れたマネジャーになるには、人に配慮できることが大切だ。
- 部下とのコミュニケーションを楽しんで、彼らの気持ちを尊重できれば、業務以外を気にしないタスク志向型の上司よりも仕事の効率がずっとよくなる。
